# Telegram AI Assistant with n8n

AI-powered Telegram Assistant built using n8n, Google Gemini, and Telegram Bot API.

## Workflow Screenshot

![Workflow](workflow.png)

## Project Overview

This project allows users to chat with an AI assistant directly through Telegram.

The workflow automatically:

1. Receives messages from Telegram
2. Sends user input to Google Gemini
3. Processes the AI response
4. Sends the answer back to Telegram

## Technologies Used

- n8n
- Google Gemini API
- Telegram Bot API
- Docker
- GitHub
